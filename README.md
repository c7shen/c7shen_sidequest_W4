## Project Title

Side Quest Week 4: Maze with Random Obstacles

---

## Description

This project is a tile-based maze game where the player navigates through levels to reach a goal tile. The user sees a top-down maze made of walls, floor tiles, a player character, a goal tile, and one randomly placed obstacle per level.

---

## Interaction Instructions

- Use WASD or arrow keys to move the player up, down, left, or right
- Player cannot move through walls and obstacles
- Reach the highlighted goal tile to advance to next level
